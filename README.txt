é necessário ter o php instalado e um projeto já com laravel criado 
no arquivo .env é necessário alterar as seguintes configurações apontando para o banco de dados 

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=aula
DB_USERNAME=root
DB_PASSWORD=

os arquivos view.html e Hello World.php estão no diretório resources\views 

view.html contem o CRUD
Hello World.php apenas um echo para exemplo
